- 로또 구입 금액을 입력하면 구입 금액에 해당하는 로또를 발급해야 한다.
- 로또 1장의 가격은 1000원이다.


## 기능 목록

- 로또 금액 입력받기
- 금액만큼의 로또 발급
- 로또 번호 자동 생성
  - 45까지
- 당첨 번호 입력
- 보너스 볼 입력
- 당첨 통계 구하기
- 수익률 계산하기
